How BFS Matrix Works

📌 Placement: New users fill the oldest open slot in the entire tree — guaranteed Breadth-First Search order using the binary heap formula: parent[i] = allUsers[(i-1)/2].


💰 Fee: 0.00001 tBNB → 100% to your BFS matrix parent. If your placement triggers a cycle on the parent, payment overflows to their parent automatically.


🔄 Auto-Recycle: When a node fills both slots, it resets and re-enters a priority recycle queue. Future placements go to the recycle queue first, then the main BFS order.

  1. Connect your MetaMask wallet
